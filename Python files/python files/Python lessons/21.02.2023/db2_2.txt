1. Вывести фамилии заказчиков в алфавитном порядке.
SELECT NAME
FROM ZAKAZ
ORDER BY NAME ASC

2. Вывести список кодов продавцов в обратном порядке.
SELECT KOD
FROM ZAKAZ
ORDER BY KOD DESC


3. Вывести все фамилии и товар заказчиков в обратном порядке (сортировка по фамилии), если фамилия входит в диапазон [С-Л].
SELECT NAME, Prod
FROM ZAKAZ
WHERE NAME GLOB '[Л-С]*'
ORDER BY NAME DESC


4. Вывести фамилии заказчиков в интервале от Па до Ле, которые не продают никакие модели  шкафов.
SELECT NAME, Prod
FROM ZAKAZ
WHERE NAME BETWEEN "Ле" AND "Па" and prod not LIKE 'шкафы'


5. Выведите фамилии продавцов, сумма заказа которых превышает 4000 и код продавцов 1003, 1010 или 1016 (три способа).
SELECT NAME, KOD
FROM ZAKAZ
WHERE SUM > 4000 AND KOD IN (1003,1010,1016)


6. Вывести коды заказчиков, которые работают в областных центрах (Иркутск,  Ижевск, Москва).


7. Выведите в алфавитном порядке список ремарок, которые есть на фирме и определены (без значения NULL).
SELECT REM
FROM ZAKAZ
WHERE REM IS NOT NULL
ORDER BY REM


8. Вывести коды всех продавцов заказчики, которых работают в городе Москва и не живут там.

SELECT KOD
FROM ZAKAZ
WHERE CITY !='Москва' AND CITY2='Москва'
9. Выведите в обратном порядке фамилии всех заказчиков, которых обслуживает продавец с номером 1012.


10. Определите всех заказчиков, которые живут во Владимире, и не покупали кухонные стенки.

SELECT NAME
FROM ZAKAZ
WHERE CITY ='Владимир' AND Prod !='кухонные стенки'

11. Определите в магазинах, каких городов продаются диваны этой фирмы. (Вывести атрибуты характеризующие город, товар и место продажи).

SELECT Prod, CITY, CITY2
FROM ZAKAZ
WHERE Prod = 'диваны' AND REM = 'Магазин'

12. Вывести на первом месте код продавца, на втором фамилию заказчика. Код должен быть упорядочен в прямом порядке, а фамилии в обратном.

SELECT KOD, NAME
FROM ZAKAZ
ORDER BY NAME DESC 


13. Напишите запрос, который выведет коды всех продавцов заказчики, которые живут в Химках и покупают стулья.
SELECT KOD
FROM ZAKAZ
WHERE CITY = 'Химки' AND Prod ='стулья'

14. Определите города, в которых продают стулья или столы для ПК.
SELECT CITY
FROM ZAKAZ
WHERE Prod ='столы для ПК' OR Prod = 'стулья'


15. Определите коды продавцов, работающих по доверенности с заказчиками, живущими в Москве и имеющими рейтинг менее  190 пунктов.

SELECT KOD
FROM ZAKAZ
WHERE REM = 'доверенность' and CITY = 'Москва' AND RATING < 190
